# google_sheets_api_tuntus
Desafio Tunts Google Sheet API

#Run
to Run this project you will need 

[Node & NPM](https://nodejs.org/en/)

[Yarn](https://yarnpkg.com/) (Optional)

Clone the project and run the following commands
**NPM**

```bash
npm install && npm start
```

**Yarn**

```bash
yarn && yarn start
```

The first time you run the application Google will display a link in terminal to authenticate and create the file `token.json`. Follow the steps to proceed
with authentication.

Once the authentication process is complete, the application will run automatically.
